The Guest Registration Notification mod displays a red, bordered box in the index as default. There are also options to display these boxes in the board index or above posts. Configure the mod through the admin control panel. This mod is compatible for both SMF 1.1.x and SMF 2.0 versions.

Version 1.5.1.
